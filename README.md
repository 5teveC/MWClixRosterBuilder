# Mech Battle Calculator

A free, fan-made tool for tracking unit stats, rules, and special abilities for tabletop mech combat games. This calculator is designed to streamline gameplay by automating heat tracking, damage tables, and special rule lookups. 
This project is not affiliated with or endorsed by any game publisher. All names, terms, and mechanics from other works remain the property of their respective owners.

---
##  Demo

Live demo: [mech-warrior-calc.vercel.app](https://mech-warrior-calc.vercel.app)

## Features

- **Damage & Heat Tracking**  
  Update and view your unit’s current damage, heat, and operational status in real time.

- **Special Ability Lookup**  
  Displays the effects of unit abilities with easy-to-read summaries and color-coded icons.

- **Custom Rules Data**  
  Rules are stored in a JSON format so you can add, remove, or modify entries for your own campaigns.

- **Responsive Interface**  
  Works on desktop and mobile devices for use at the table.
